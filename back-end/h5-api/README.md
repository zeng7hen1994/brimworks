## The api for Luban-H5 powered by [Strapi](https://github.com/strapi/strapi/)

## Project setup

```bash
# install strapi globally
npm install strapi@beta -g

yarn install

strapi develop

# default database is sqlite3(h5-api/.tmp/data.db)
# visit http://locahost:1337/admin
```
